# Week 2 - Creating and Configuring a Maven Project

## Objective
Create and configure a Maven project for a Library Management application and add the required Spring dependencies.

## Files
- pom.xml

## Dependencies Included
- Spring Context
- Spring AOP
- Spring Web MVC

## Maven Plugin
- Maven Compiler Plugin
- Java Version: 1.8

## Description
This exercise demonstrates how to create a basic Maven project and configure it by adding Spring Framework dependencies and the Maven Compiler Plugin. The project is set to compile using Java 1.8.
