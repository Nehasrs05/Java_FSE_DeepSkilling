# Week 2 - Create a Spring Web Project using Maven

## Objective

Create a basic Spring Boot Web project using Maven.

## Files

- SpringLearnApplication.java
- pom.xml
- application.properties

## Technologies Used

- Java
- Spring Boot
- Spring Web
- Maven

## Description

This exercise demonstrates the creation of a Spring Boot Web application using Maven. The project includes the required Spring Web and Spring Boot DevTools dependencies. The main application starts the Spring Boot application and logs a message to verify that the `main()` method is executed successfully.

## Main Components

- `SpringLearnApplication.java` – Entry point of the application.
- `pom.xml` – Maven configuration with Spring Boot dependencies.
- `application.properties` – Application and logging configuration.
