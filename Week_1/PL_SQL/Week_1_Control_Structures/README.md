# Week 1 - PL/SQL - Control Structures

## Objective

Practice PL/SQL control structures using loops, conditions, and updates.

## Scenarios

### Scenario 1
Apply a 1% discount to loan interest rates for customers above 60 years.

### Scenario 2
Promote customers with a balance greater than $10,000 to VIP status.

### Scenario 3
Print reminder messages for loans due within the next 30 days.

## Files

- Exercise1_Control_Structures.sql

## PL/SQL Concepts Used

- DECLARE
- BEGIN...END
- FOR LOOP
- UPDATE
- WHERE clause
- COMMIT
- DBMS_OUTPUT.PUT_LINE
- SYSDATE
