# Week 1 - Arrange-Act-Assert (AAA) Pattern, Setup and Teardown

## Objective

Learn how to organize JUnit test cases using the Arrange-Act-Assert (AAA) pattern and use setup and teardown methods.

## Files

- Calculator.java
- CalculatorTest.java

## Concepts Covered

- AAA Pattern
  - Arrange
  - Act
  - Assert

- @Before Annotation
  - Executes before every test method.

- @After Annotation
  - Executes after every test method.

## Description

This exercise demonstrates how to prepare test data before execution, perform the required action, verify the result using assertions, and clean up resources after the test execution.
