# Week 1 - E-commerce Platform Search Function

## Objective
Implement Linear Search and Binary Search for searching products in an e-commerce platform.

## Files
- Product.java
- SearchAlgorithms.java
- SearchTest.java

## Big O Analysis

### Linear Search
- Best Case: O(1)
- Average Case: O(n)
- Worst Case: O(n)

### Binary Search
- Best Case: O(1)
- Average Case: O(log n)
- Worst Case: O(log n)

## Which is Better?

Binary Search is more efficient for large, sorted datasets because it reduces the search space by half in each iteration. Linear Search is suitable for small or unsorted collections.
