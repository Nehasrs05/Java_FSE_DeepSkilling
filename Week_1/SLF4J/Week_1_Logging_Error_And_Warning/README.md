# Week 1 - Logging Error Messages and Warning Levels using SLF4J

## Objective

Demonstrate logging of error and warning messages using the SLF4J logging framework with Logback.

## Files

- pom.xml
- LoggingExample.java

## Dependencies

- SLF4J API (1.7.30)
- Logback Classic (1.2.3)

## Description

This exercise demonstrates how to configure SLF4J with Logback and use it to log messages at different severity levels.

### Log Levels Demonstrated

- ERROR
- WARN

## Sample Output

```
ERROR - This is an error message
WARN  - This is a warning message
```
