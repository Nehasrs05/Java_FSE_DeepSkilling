# Week 1 - Mocking and Stubbing using Mockito

## Objective

Learn how to use Mockito to mock external dependencies and stub method responses during unit testing.

## Files

- ExternalApi.java
- MyService.java
- MyServiceTest.java

## Concepts Covered

- Mockito
- Mock Objects
- Stubbing
- Unit Testing
- Dependency Injection

## Description

This exercise demonstrates how to create a mock object for an external API using Mockito. The mocked API returns predefined data using stubbing, allowing the service class to be tested independently of the actual external system.
